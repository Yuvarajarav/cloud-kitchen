# Cloud Kitchen Web Application

A full-stack web application developed for managing online food orders in a cloud kitchen environment. The system allows users to browse food items, add them to a cart, and place orders. Once an order is confirmed, the restaurant owner receives the order details via email.

## Features

- User login and signup authentication
- Browse available food menu
- Add food items to cart
- Place and confirm orders
- Email notification to the restaurant owner
- Order summary display after confirmation

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- PHPMailer

## System Workflow

1. Users create an account or login to the system.
2. Users browse the food menu available in the cloud kitchen.
3. Selected items can be added to the cart.
4. Users provide their delivery address and confirm the order.
5. PHPMailer sends an automated email containing the order details to the restaurant owner.

## Database
The system uses a MySQL database to store:

- User account details
- Food menu information
- Order details
- User delivery address

## Installation

1. Install XAMPP or any local server environment.
2. Place the project folder inside the `htdocs` directory.
3. Import the MySQL database using phpMyAdmin.
4. Configure database connection settings in the PHP files.
5. Start Apache and MySQL in XAMPP.
6. Open the project in a browser.
